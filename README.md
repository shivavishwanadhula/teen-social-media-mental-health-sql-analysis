# 🧠 Teen Social Media & Mental Health Analysis Using SQL

## 📌 Project Overview

This project analyzes the relationship between teen social media usage and mental health using advanced SQL techniques. The analysis focuses on behavioral patterns, addiction levels, stress levels, sleep duration, academic performance, and depression indicators.

The project was developed as a real-world SQL analytics case study to demonstrate practical database analysis and business intelligence skills.

---

# 🎯 Project Objectives

- Analyze student social media behavior
- Identify high-risk students using addiction and stress metrics
- Study the relationship between sleep habits and academic performance
- Generate business insights using SQL
- Implement automation using triggers and stored procedures

### 📊 Key Business Insights

* Students with higher addiction levels showed lower academic performance.
* Reduced sleep duration increased stress and mental health risks.
* Certain social media platforms showed higher stress levels.
* Trigger-based alert systems can identify high-risk students automatically.
* SQL analytics can support early mental health intervention systems.


### 💡 Business Recommendations
* Provide counseling support for high-risk students.
* Encourage healthy sleep habits and screen-time management.
* Monitor excessive social media usage patterns.
* Build automated dashboards for early mental health detection.
* Use SQL-based alert systems for proactive intervention.
